## Form fields

The `Form` supports 2 types of inputs: `Form.TextInput` and `Form.TextArea`

<Playground>
  <Form style={{ width: "50%" }}>
    <Form.TextInput
      label="Title"
      placeholder="Enter title"
      helpText="This is a snippet of text to help the user"
    />
    <Form.TextArea label="Description" placeholder="Enter descrtiption" />
  </Form>
</Playground>

## HelpText

All fields support a `helpText` field

<Playground>
  <Form style={{ width: "50%" }}>
    <Form.TextInput
      label="Title"
      placeholder="Enter title"
      helpText="This is a snippet of text to help the user"
    />
  </Form>
</Playground>

## Optional fields

You can mark fields as `optional`

<Playground>
  <Form style={{ width: "50%" }}>
    <Form.TextArea label="Description" optional />
  </Form>
</Playground>

## Label Definition

Add a `labelDefinition` to explain what the field means

<Playground>
  <Form style={{ width: "50%" }}>
    <Form.TextInput
      label="Tags"
      labelDefinition="Tags help users search by topic"
    />
  </Form>
</Playground>

## Length limit

You can add a length `counter` which indicates the number of charachters the user can add more

<Playground>
  <Form style={{ width: "50%" }}>
    <Form.TextArea label="Description" counter={160} />
  </Form>
</Playground>
